# Кастомные звуки для Holy Ground

## Необходимые аудиофайлы

Поместите следующие WAV файлы в эту папку:

1. **bell_chime.wav** - Звук колокола для лечения (каждые 2 секунды)
   - Рекомендации: Чистый звон колокола, 1-2 секунды длительность
   - Частота: 44.1 kHz, 16-bit

2. **holy_cast.wav** - Звук применения способности
   - Рекомендации: Мощный святой/магический звук каста
   - Длительность: 0.5-1 секунда

3. **holy_ambient.wav** - Фоновый звук святой зоны
   - Рекомендации: Тихий святой/ангельский фон, зацикленный
   - Длительность: 3-5 секунд для зацикливания

## Форматы аудио

- **Формат:** WAV (несжатый)
- **Частота дискретизации:** 44.1 kHz
- **Разрядность:** 16-bit
- **Каналы:** Моно или стерео

## Примеры источников звуков

- Бесплатные звуки: freesound.org, zapsplat.com
- Игровые звуки: Warcraft, Diablo, Sacred
- Сгенерированные: Audacity, Reaper

## Настройка громкости

Громкость звуков можно настроить в файлах:
- `holy_ground_bell.vsnd` - громкость колокола
- `holy_ground_cast.vsnd` - громкость каста  
- `holy_ground_ambient.vsnd` - громкость фона

Текущие настройки:
- Bell: 0.8 (80%)
- Cast: 0.7 (70%) 
- Ambient: 0.5 (50%)
