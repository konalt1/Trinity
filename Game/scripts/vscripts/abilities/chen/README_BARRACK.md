# Механика Барака Чена

## Описание
Ультимейт Чена позволяет превратить прирученного крипа в барак, который производит юнитов из семейства этого крипа за золото.

## Основная механика

### Создание барака
- **Способность**: `chen_barrack` (Ultimate)
- **Цель**: Прирученный крип (союзный крип под контролем Чена)
- **Эффект**: Крип превращается в барак с HP = HP крипа + бонус от уровня способности
- **Ограничение**: Максимальное количество бараков = уровень ульты (1/2/3, с аганимом = 4)
- **Поведение**: При превышении лимита самый старый барак уничтожается с возвратом золота

### Производство крипов
Барак имеет 4 способности для производства крипов разных уровней:

1. **Tier 1** (`chen_barrack_spawn_t1`)
   - Стоимость: 75 золота
   - Время производства: 10 секунд
   - Доступно: С 1 уровня ульты

2. **Tier 2** (`chen_barrack_spawn_t2`)
   - Стоимость: 150 золота
   - Время производства: 18 секунд
   - Доступно: Со 2 уровня ульты

3. **Tier 3** (`chen_barrack_spawn_t3`)
   - Стоимость: 275 золота
   - Время производства: 30 секунд
   - Доступно: С 3 уровня ульты

4. **Ancient** (`chen_barrack_spawn_ancient`)
   - Стоимость: 500 золота
   - Время производства: 50 секунд
   - Доступно: Только с Aghanim's Scepter

### Система очередей
- При нажатии на покупку золото **сразу списывается** с Чена
- Барак начинает производство крипа (показывается модификатор с таймером)
- После завершения производства крип появляется рядом с бараком
- Барак автоматически начинает производство следующего крипа из очереди
- **Максимум в очереди**: 5 юнитов (1 в производстве + 4 в очереди)

### Уничтожение барака
- При уничтожении барака **всё неизрасходованное золото возвращается**
- Золото делится поровну между всеми игроками команды Чена
- Барак можно уничтожить вручную способностью `chen_barrack_self_destruct`:
  - Наносит 300 урона в радиусе 400
  - Возвращает неизрасходованное золото

## Прогрессия по уровням

### Уровень 1 ульты
- Максимум бараков: **1**
- Доступные тиры: **T1**
- Бонус HP барака: 700
- Минимальный HP барака: 1200

### Уровень 2 ульты
- Максимум бараков: **2**
- Доступные тиры: **T1, T2**
- Бонус HP барака: 1100
- Минимальный HP барака: 1700

### Уровень 3 ульты
- Максимум бараков: **3**
- Доступные тиры: **T1, T2, T3**
- Бонус HP барака: 1500
- Минимальный HP барака: 2200

### С Aghanim's Scepter
- Максимум бараков: **4**
- Доступные тиры: **T1, T2, T3, Ancient**
- Разблокирует производство эншентов

## Технические детали

### Определение семьи крипа
Система автоматически определяет семью крипа по его имени (unit name):
- Ищет совпадения в таблице `CHEN_BARRACK_FAMILIES`
- Если семья не найдена, использует generic юнитов (melee/ranged/siege)

### Fallback юниты
Если юнит из семьи не найден, используются запасные варианты:
- **T1**: `npc_chen_barrack_melee`
- **T2**: `npc_chen_barrack_ranged`
- **T3**: `npc_chen_barrack_siege`
- **Ancient**: `npc_chen_barrack_siege`

### Возврат золота
При уничтожении барака:
1. Подсчитывается зарезервированное золото (сумма всех заказов в очереди)
2. Золото делится поровну между всеми героями команды Чена
3. Остаток от деления получает первый герой в списке

### Таланты
Система поддерживает таланты Чена:
- `special_bonus_unique_custom_chen_7`: Бонус HP барака
- `special_bonus_unique_custom_chen_8`: Снижение стоимости и времени производства

## Визуальные эффекты
- **Создание барака**: Пыль трансформации + эффект возрождения Aegis
- **Производство крипа**: Пыль трансформации + звук строительства
- **Завершение производства**: Звук завершения строительства + телепорт
- **Самоуничтожение**: Взрыв мины Techies
