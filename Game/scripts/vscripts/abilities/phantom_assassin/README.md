# Phantom Assassin - Blur

## Описание
Способность Blur для Phantom Assassin, переписанная в Lua.

## Функциональность

### Пассивная часть
- **Уклонение**: Предоставляет базовое уклонение от атак (20/25/30/35%)
- **Невидимость**: Делает героя невидимым, когда рядом нет вражеских героев в радиусе 1600 единиц
- **Автоматическое скрытие**: Постоянно проверяет наличие врагов и скрывает/показывает героя

### Активная часть
- **Усиленное уклонение**: При активации увеличивает уклонение до 50/60/70/80%
- **Полная невидимость**: Делает героя полностью невидимым на 8 секунд
- **Мгновенная активация**: Способность активируется мгновенно без задержки

## Файлы
- `phantom_assassin_blur.lua` - Основной скрипт способности
- `phantom_assassin_blur.txt` - Данные способности (для справки)

## Модификаторы
- `modifier_phantom_assassin_blur` - Пассивный модификатор
- `modifier_phantom_assassin_blur_active` - Активный модификатор

## Значения способности
- **evasion_chance**: 20/25/30/35% (пассивное уклонение)
- **radius**: 1600 (радиус обнаружения врагов)
- **active_duration**: 8 секунд (длительность активной способности)
- **active_evasion_chance**: 50/60/70/80% (активное уклонение)

## Таланты
- **special_bonus_unique_phantom_assassin_2**: +15% к пассивному уклонению
- **special_bonus_unique_phantom_assassin_3**: +4 секунды к длительности активной способности
- **special_bonus_unique_phantom_assassin_4**: +20% к активному уклонению

## Использование
Способность автоматически применяется как пассивная. Для активации усиленной версии используйте способность как активную (клавиша по умолчанию).
